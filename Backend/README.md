GET

http://localhost:5555/products --> Devuelve todos los productos

http://localhost:5555/products/1 --> Devuelve el producto segun ID

POST

http://localhost:5555/users --> Crea un nuevo usuario

http://localhost:5555/sales --> Crea una nueva venta

PUT

http://localhost:5555/products/1 --> Actualiza el valor de algun producto segun ID

DELETE

http://localhost:5555/users/1 --> Elimina a un usuario segun ID
